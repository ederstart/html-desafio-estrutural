Entendendo a Estrutura das Tags HTML

As tags HTML são elementos que compõem a estrutura de um website. Elas são responsáveis por definir o conteúdo e a formatação de uma página web.

Títulos e Textos

Ao criar um título em HTML, você pode utilizar as tags <h1> até <h6>. Sendo que <h1> é o título mais importante e <h6> é o menos importante. Ao utilizar essas tags, você estará definindo a hierarquia dos títulos em sua página. Por exemplo: o título principal da página deve ser definido com a tag <h1>, enquanto os subtítulos podem ser definidos com as tags <h2>, <h3>, <h4>, <h5> e <h6>.

Para criar um parágrafo, você pode utilizar a tag <p>. Se deseja dar destaque ao texto, pode utulizar a tag <strong> para negrito e <mark> para destacar. Por exemplo: texto em negrito e texto destacado. Se deseja criar um texto em italico pode utilizar a tag <em> ou a tag <i>. Por exemplo: texto em italico e texto em italico.

Listas e Estruturação

Para criar uma lista não ordenada, você pode utilizar a tag <ul> e para criar uma lista ordenada, você pode utilizar a tag <ol>. Dentro dessas tags, você pode utilizar a tag <li> para criar os itens da lista.

Por exemplo: todas as tags explicadas nas aulas.

Lista Não Ordenada
  • <h1> até <h6>
  • <p>
  • <mark>
  • <small>
  • <i>
  • <u>
  • <strong>

As listas não ordenadas são utilizadas para criar listas de itens que não possuem uma ordem específica.

Lista Ordenada
  1. <ol>
  2. <ul>
  3. <li>
  4. <a>
  5. <hr>
  6. <sub>
  7. <sup>
  8. <blockquote>

As listas ordenadas são utilizadas para criar listas de itens que possuem uma ordem específica.

Hyperlinks

A tag <a> é utilizada para criar links em uma página web. Por exemplo: Google. Voê pode utilizar o atributo href para definir o destino e o atributo target _blank para definir se o link será aberto em uma nova aba. Ou manter o padrão de abertura na mesma aba.

Citações e Destaques

Para criar uma citação em HTML, você pode utilizar a tag <q> para citações curtas e a tag <blockquote> para citações longas.

A tag <abbr> é utilizada para criar abreviações em uma página web. Por exemplo: CSS. Você pode utilizar o atributo title para definir o significado da abreviação.

A tag <del> serve para riscar um texto. Por exemplo: texto riscado.
